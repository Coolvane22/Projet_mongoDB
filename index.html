Présentation du projet

Nous sommes 2 étudiantes à l'université Rennes 2, en première année de master MAS (Mathématiques Appliquées Statistiques). Dans le cadre du cours de Base de données NoSQL, nous devons réaliser un projet qui porte sur la manipulation des données issues de bases MongoDB pour générer des visualisations.
Pour ce faire, nous avons à disposition 3 bases de données que nous présenterons ci-dessous. Les visualisations concernant la première base ont été réalisées via des requêtes MongoDB avec Python. Le rendu a été généré avec Dash (ou Bokeh) et est visible via le menu de navigation (onglet Site Python).
Les visualisations concernant les 2 autres bases ont été réalisées via des requêtes MongoDB avec R. Le rendu a été généré avec RShiny et est visible via le menu de navigation (onglet Site R).
