Présentation du projet

Nous sommes 2 étudiantes à l'université Rennes 2, en première année de master MAS (Mathématiques Appliquées Statistiques). Dans le cadre du cours de Base de données NoSQL, nous devons réaliser un projet qui porte sur la manipulation des données issues de bases MongoDB pour générer des visualisations.
Pour ce faire, nous avons à disposition 3 bases de données que nous présenterons ci-dessous. Les visualisations concernant la première base ont été réalisées via des requêtes MongoDB avec Python. Le rendu a été généré avec Bokeh et est visible via le menu de navigation (onglet Site Python).
Les visualisations concernant les 2 autres bases ont été réalisées via des requêtes MongoDB avec R. Le rendu a été généré avec RShiny et est visible via le menu de navigation (onglet Site R).

Présentation des bases de données

La première base de données, intitulée 'docotlib', contient des informations (récupérées en janvier 2022) relatives aux centres de vaccinations contre la Covid-19 en Bretagne. Nous nous sommmes focalisées sur les centres de vaccinations situés à moins de 50km de Rennes.
La deuxième base de données, intitulée 'publications', contient des informations sur les publications de scientifiques du laboratoire IRISA pour l'année 2021. Elles ont été extraites du service HAL. Nous nous sommes focalisées sur les 20 auteurs qui ont participés à l'écriture du plus grand nombre d'articles.
La troisième et dernière base de données, intitulée 'food', contient des informations sur les restaurants des cinq arrondissements de New York (Manhattan, Brooklyn, Queens, le Bronx et Staten Island).
